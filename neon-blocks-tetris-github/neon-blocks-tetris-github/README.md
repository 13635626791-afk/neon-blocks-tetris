# Neon Blocks · 经典俄罗斯方块

一个支持电脑键盘和手机触控的经典俄罗斯方块网页游戏。

## 功能

- 标准 10 × 20 棋盘与 7-Bag 随机器
- 下一个方块队列、暂存方块和幽灵落点
- 消行计分、等级加速与连击奖励
- 键盘与触屏操作
- 暂停、重新开始、音效开关
- 本机保存最高分

## 操作

| 操作 | 按键 |
| --- | --- |
| 左右移动 | `←` / `→` |
| 软降 | `↓` |
| 顺时针旋转 | `↑` / `X` |
| 逆时针旋转 | `Z` |
| 硬降 | `Space` |
| 暂存 | `C` |
| 暂停 | `P` / `Esc` |
| 重新开始 | `R` |

仓库包含 GitHub Pages 自动部署配置。代码推送到 `main` 分支后会自动发布。
